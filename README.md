# Webpack-starter
Este es un proyecto inicial para crear aplicaciones web con webpack.
### Notas:
Recuerde reconstruir los modulos de Node JS
....
npm install
....
Y para reconstruir la carpeta dist
....
npm run build
....
Para el ambiente de desarrollo
....
npm run build:dev
....
